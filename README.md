# Análisis exploratorio de vehículos con Streamlit

Este proyecto consiste en una aplicación web interactiva desarrollada con **Streamlit**, que permite realizar un **análisis exploratorio de datos (EDA)** sobre un conjunto de datos de vehículos usados.

## Funcionalidad de la aplicación

La aplicación web proporciona las siguientes funcionalidades:

- Visualización de una muestra del conjunto de datos de vehículos.
- Generación de un **histograma interactivo** que muestra la distribución de precios de los vehículos.
- Generación de un **gráfico de dispersión interactivo** que muestra la relación entre el precio y el kilometraje de los vehículos.
- Interacción mediante botones que permiten al usuario decidir qué visualizaciones mostrar.

Las visualizaciones se crean utilizando **Plotly**, lo que permite una exploración dinámica de los datos.

## Tecnologías utilizadas

- Python
- Streamlit
- Pandas
- Plotly
